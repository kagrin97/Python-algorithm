이 강의에서 배운 내용을 검토해 보겠습니다. <!+tab을 누르면 자동완성>

1.<!DOCTYPE html>선언은 항상 HTML 파일에서 코드의 첫 번째 줄 수 있어야합니다. 이를 통해 브라우저는 예상되는 HTML 버전을 알 수 있습니다.

2.<html>요소는 HTML 코드를 모두 포함됩니다.

3.제목과 같은 웹 페이지에 대한 정보는 페이지 내에 속합니다 <head>.

4.<title>헤드 내부의 요소 를 사용하여 웹 페이지에 제목을 추가 할 수 있습니다 .

5.웹 페이지의 제목이 브라우저의 탭에 나타납니다.

6.앵커 태그 ( <a>)는 내부 페이지, 외부 페이지 또는 동일한 페이지의 콘텐츠에 연결하는 데 사용됩니다.     ##<a href = "url">클릭한 페이지로 이동</a>##

7.웹 페이지에 섹션을 만들고 <a>태그를 사용하여 섹션으로 이동하고 이동 id하려는 요소에를 추가 할 수 있습니다.

8.HTML 요소 사이의 공백은 브라우저에 요소가 표시되는 방식을 변경하지 않으면 서 코드를 더 쉽게 읽을 수 있도록합니다.

9.들여 쓰기를 사용하면 코드를 더 쉽게 읽을 수 있습니다. 부모-자식 관계를 볼 수 있습니다.

10.주석은 다음 구문을 사용하여 HTML로 작성됩니다 <!-- comment -->..

생성 한 작업 공간을 편집하고 어떻게 변경되는지 관찰하십시오!
