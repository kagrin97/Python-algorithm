css 명령어 id = # , class =.

1.CSS는 HTML 요소의 모양을 변경할 수 있습니다. 이를 위해 CSS는 HTML 요소를 선택한 다음 스타일을 적용해야합니다.

2.CSS는 태그, 클래스 또는 ID별로 HTML 요소를 선택할 수 있습니다.

3.하나의 HTML 요소에 여러 CSS 클래스를 적용 할 수 있습니다.

4.클래스는 재사용이 가능하지만 ID는 한 번만 사용할 수 있습니다.

5.ID는 클래스보다 더 구체적이고 클래스는 태그보다 더 구체적입니다. 즉, ID는 클래스의 모든 스타일을 재정의하고 클래스는 태그 선택기의 모든 스타일을 재정의합니다.

6.여러 선택기를 함께 연결하여 요소를 선택할 수 있습니다. 이것은 특이성을 높이지만 필요할 수 있습니다.

7.선택기를 공백으로 구분하여 중첩 된 요소를 선택할 수 있습니다.

8.관련되지 않은 여러 선택자는 선택기 이름을 쉼표로 구분하여 동일한 스타일을받을 수 있습니다.

h2.destination {
  font-family: cursive;
}   (위 명령어는 h2가 포함되고 destination도 포함되는 클래스만 폰트가 바뀜 교집합느낌)

.description h5 {
  color: teal;
} (위 명령어는 .description이 먼저포함된 클래스중에 h5 만 컬러가 바뀜 앞뒤로 바꾸면 결과가 달라짐)

h5,
p {
  font-family: Georgia;
} (이렇게 쓸경우 h5,p 둘다 모두 폰트가바뀜 합집합 느낌)
