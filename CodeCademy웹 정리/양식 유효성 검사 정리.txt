2.required입력 관련 요소에 속성을 추가하면 입력 필드에 정보가 있는지 확인합니다.

3.min숫자 입력 요소 의 속성에 값을 할당 하면 허용 가능한 최소값의 유효성을 검사합니다.

4.max숫자 입력 요소 의 속성에 값을 할당 하면 허용 가능한 최대 값의 유효성을 검사합니다.

5.minlength텍스트 입력 요소 의 속성에 값을 할당 하면 허용 가능한 최소 문자 수의 유효성을 검사합니다.

6.maxlength텍스트 입력 요소 의 속성에 값을 할당 하면 허용 가능한 최대 문자 수의 유효성을 검사합니다.

7.pattern입력을 제공된 정규식과 일치하도록 정규식을 할당합니다 .
